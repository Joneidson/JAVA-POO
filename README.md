# 💻 JAVA-POO 🧠

Projeto em Java para praticar **programação orientada a objetos (POO)**.  
Desenvolvido com foco em conceitos fundamentais como:

- 🧱 **Classes**  
- 👤 **Objetos**  
- 🔐 **Encapsulamento**  
- 🧬 **Herança**  
- 🎭 **Polimorfismo**  
- 🗂️ **Estrutura de pacotes**

---

## 🚀 Tecnologias Utilizadas

- ☕ **Java 24+**
- 🧠 **IntelliJ IDEA**
- 🗃️ **Git & GitHub**
